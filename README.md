# InvesthreeAI

InvesthreeAI 是一个基于人工智能的投资分析平台，旨在帮助投资者做出更明智的投资决策。

## 功能特点

- 智能投资分析
- 实时市场数据
- 个性化投资建议
- 投资组合管理

## 技术栈

- TypeScript
- React
- Node.js
- AI/ML 技术

## 安装与运行

1. 克隆项目
```bash
git clone [项目地址]
```

2. 安装依赖
```bash
npm install
```

3. 运行开发服务器
```bash
npm run dev
```
